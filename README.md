
# AccelSystems Line Follower V2
### 3D Printed version for ASLFV2

# HARDWARE
- Raspberry Pi Pico W
- HCRS04
- Pololu QTRX 13x ARRAY
- 2x 1000RPM motors
- Pololu md18a motor driver
- LM7805 VOLTAGE REGULATOR (CUSTOM MADE PCB)
- 9.6V 1.6Ah NiMH Battery

# DATA
- Print weights: 38g
- Material used: GRAY PETG / RED PLA
- CPU SPEED: 133Mhz
- Wheel diamater: 32mm + tire / 15mm disc
- Wheel rubber used: LEGO RUBBER

# SOFTWARE USED
- Fusion360
- Thonny
- MicroPython > [documentation](https://micropython.org/download/RPI_PICO_W/)
